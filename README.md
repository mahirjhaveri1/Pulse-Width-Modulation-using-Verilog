# Pulse Width Modulator (PWM) in Verilog

## Overview
This project implements a parameterized Pulse Width Modulator (PWM) using Verilog HDL. The design generates a PWM signal based on a configurable duty cycle input.

## Features
- Parameterized design (configurable resolution)
- Supports variable duty cycle control
- Simple and synthesizable architecture
- Testbench included for simulation

## Design Description
The PWM signal is generated by comparing a counter with the duty cycle input:
- If counter < duty_cycle → Output HIGH
- Else → Output LOW

## Files
- pwm_generator.v : Main PWM module
- pwm_tb.v : Testbench for simulation

## Simulation
The testbench verifies PWM output for:
- 25% duty cycle
- 50% duty cycle
- 75% duty cycle
- 100% duty cycle

## How to Run
1. Open in any Verilog simulator (ModelSim / Vivado / Icarus Verilog)
2. Compile both files
3. Run simulation
4. Observe waveform

## Applications
- Motor speed control
- LED brightness control
- Power electronics
