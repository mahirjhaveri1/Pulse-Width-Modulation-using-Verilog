# Pulse Width Modulator (PWM) in Verilog

## Overview
This project implements a Pulse Width Modulator (PWM) using Verilog HDL. The design generates a PWM signal based on a configurable duty cycle input using a counter-based approach.

## Features
- Fixed 8-bit resolution (0–255 range)
- Supports variable duty cycle control
- Simple and fully synthesizable design
- Testbench included for simulation

## Design Description
The PWM signal is generated by comparing a counter with the duty cycle input:
- If counter < duty_cycle → Output HIGH
- Else → Output LOW

An 8-bit counter continuously increments on each clock cycle and resets automatically after overflow, creating a repeating PWM waveform.

## Files
- pwm_generator.v : Main PWM module

## Simulation
The testbench verifies PWM output for:
- 25% duty cycle (~64)
- 50% duty cycle (~128)
- 75% duty cycle (~192)
- 100% duty cycle (255)

## How to Run
1. Open in any Verilog simulator
2. Compile both files
3. Run simulation
4. Observe waveform

## Applications
- Motor speed control
- LED brightness control
- Power electronics
